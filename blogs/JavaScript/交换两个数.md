### 临时变量存储
```js
let a = 0
let b = 1
let c
c = b // 1
b = a // 0
a = c // 1
```

### 加减运算（只能针对数字）
```js
let a = 0
let b = 1
a = a + b // 1
b = a - b // 0
a = a - b // 1
```

### 利用数组
```js
let a = 0
let b = 1
a = [b, b = a][0] // 1
```

### 解构赋值
```js
let a = 0
let b = 1
[a, b] = [b, a]
```

### 异或（针对数字）
```js
/**
 * 异或的原理:
 *  1.异或就是不相等的位为1，相等的位为0 1^0 = 1  1^1 = 0  0^0 = 0
 *  2.一个数异或自己就会等于0   x^x = 0
 *  3.一个数跟0异或就等于自身   x^0 = x
*/
let a = 0
let b = 1
a = a^b
b = a^b
a = a^b
```
